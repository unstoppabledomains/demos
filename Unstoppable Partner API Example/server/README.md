In the project directory, you should:

### Go into the server Folder and update the .env API_KEY_VALUE with your Sandbox API key from the [Unstoppable Dashboard](https://docs.unstoppabledomains.com/domain-distribution-and-management/quickstart/retrieve-an-api-key/). Use .env.example to create your .env

### `npm run dev` to start the server

Runs the app in the development mode.\
Open [http://localhost:3001](http://localhost:3001) to view it in your browser.
